Project: Stylable

Stylable is a smart outfit generator which uses machine learning, object detection, webscraping, and a custom outfit algorithm to give users ideas for their outfits.

Core Libraries - YOLO, Selenium, CustomTKinter, PIL, OpenCV

Stylable begins loading the Selenium webdriver in the background. This is to expedite the webscraping process.

The user uploads a pile of clothes, selects whether they want to see masculine or feminine outfits, and then submits. Stylable then detects which outfits are present in the image, classifying them and then inputting the present outfits into our custom algorithm. The algorithm submits all possible outfits into the Selenium webdriver, to which Stylable downloads the best fit images. Those downloaded images are then shown to the user.
